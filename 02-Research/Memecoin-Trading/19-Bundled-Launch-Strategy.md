# Bundled Launch Strategy — How Insiders Work & How to Counter

> [[00-INDEX|← Back to Index]]

---

## What Bundling Is

Insiders launch a coin and **simultaneously buy across 10–20+ wallets in the same block** before any public buyer can see it. Looks like organic early buying — it's one person controlling all of it.

**On a chart it looks like:**
1. Token launches → immediate green candle (the bundle buy)
2. Price spikes → retail sees momentum and apes in
3. Insiders dump from ALL wallets into the retail buying pressure
4. Chart collapses → retail holds the bags

> This is the most common manipulation on Pump.fun. ~18.81% of tokens use bundle buys.

---

## How to Detect It

### Tool 1: Axiom Vision
- Bundle % shown directly on every token
- **< 10%** = clean
- **10–15%** = caution
- **> 15%** = avoid
- **> 30%** = hard exit, insiders own the float

### Tool 2: Bubblemaps (bubblemaps.io)
- Paste contract address → visual map loads in seconds
- Bubbles connected by lines = same person, multiple wallets
- If top 5–10 holders all link back to one central wallet = bundled launch

### Tool 3: GMGN
- Shows bundle % and sniper % per token
- Check "first 70 buyers" tab — if they all funded from same wallet = insider

### On-Chain Tells
- All insider wallets received SOL from **same funding wallet** within 24hrs of launch
- All bought in the **same block** as token creation
- Brand new wallets — never traded anything else before this coin

## Two Ways to Trade Around Bundles

### Strategy 1 — Ride the FOMO (High Risk, Fast)
- Spot a bundled coin early
- Buy a **tiny** position as retail starts piling in
- Set take-profit at **1.5x–2x** and exit BEFORE insiders dump
- You're riding the retail FOMO wave knowing it will collapse
- **Risk:** Must be watching live. One second too slow = you're the bag holder

### Strategy 2 — Wait for the Dump (Lower Risk)
- See high bundle % → don't buy the spike
- Watch it dump after insiders exit
- If chart holds **above 50% of spike high** after dump = real organic buyers are accumulating
- Buy the recovery, not the initial spike
- This is the safer play — you let insiders dump on others, then enter

## The Real Lesson — What to Actually Buy

**Fair launch = no bundle = equal opportunity**

Pump.fun fair launches where:
- Bundle % < 10%
- Dev wallet < 5%
- Sniper % < 25%
- No wallets all funded from same source

These are where the real 10x–100x plays live because insiders aren't waiting to dump on you.
